Template Engines

A Template Engine is a software library that takes a generic template and combines it with dynamic data to generate output in some desired format. A Web Template Engine is a templating engine that generates output suitable for consumption by web clients (typically HTML or XHTML), such as, browsers. Web template engines allows develops to generate web pages dynamically by combining data with page templates.

Web Templating with Spring Boot

Spring Boot supports the following templating engines out of the box and support for more can be added easily.

  • JavaServer Pages (JSPs)
  • Thymeleaf: See example
  • Freemarker
  • Apache Velocity
  • Mustache
  • Jade4J: Through a contribution. See example
