Welcome to Spring Boot!

Spring Boot is a project under the Spring Framework umbrella for developing stand-alone Java application quickly and reliably. It combines the Spring Framework with a variety of other open-source libraries for rapidly developing and deploying Java applications using robust software design and development principles such as Domain-driven Design (DDD), Service-oriented Architecture (SOA) and Microservices.

Spring Boot Features

The following Spring Boot features are demonstrated by this application.

  • Web Templating
    • Thymeleaf Template Engine
    • Jade4J Template Engine
